# Converters — Конвертеры значений

## Обзор категории

**Converters** — классы, реализующие `IValueConverter`. Используются в XAML-привязках для преобразования данных из одного типа в другой.

### Структура папки

```
Converters/
└── BoolToWidthConverter.cs  # Конвертер bool ↔ double для ширины
```

### Роль в архитектике

```
ViewModel (bool IsExpanded) → Converter → View (double Width)
```

Конвертеры отвечают за:
- Преобразование типов для биндингов (`bool` → `double`, `enum` → `Visibility`)
- Форматирование значений (даты, числа, размеры)
- Обратное преобразование (`ConvertBack`) для двухсторонних привязок

### Правила оформления

1. **Один конвертер = один файл**
2. **Реализация `IValueConverter`** — оба метода (`Convert`, `ConvertBack`)
3. **Обработка null** — возвращать значение по умолчанию
4. **Без побочных эффектов** — конвертер не должен изменять состояние
5. **CultureInfo** — учитывать при форматировании
